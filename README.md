# Frontend with React

### Requirements:
- git
- docker
- docker-compose
- thats it

## Set up:
##### `git clone https://github.com/kritagyakhandelwal314e/react_frontend.git` <br>
##### `cd react_frontend`
##### `docker-compose up --build` <br>

## Usage
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
